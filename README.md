# 놀러 가자! 콘텐츠! 🎬

> **크리에이터를 위한 여행 장소 매칭 플랫폼**

한국관광공사 콘텐츠랩(api.visitkorea.or.kr)의 관광정보 **50,700건**을 수집·분석하여
크리에이터가 원하는 **지역**과 **감정**에 맞는 여행 장소를 매칭해주는 Streamlit 앱입니다.

---

## 🗂️ 파일 구조

```
├── app.py                        # Streamlit 메인 앱
├── crawl_visitkorea.py           # 관광정보 크롤러 (hub/getTourDbInfo.do API)
├── add_columns.py                # 지역명·감정 컬럼 추가 스크립트
├── visitkorea_tour_data_v2.csv   # 최종 데이터 (50,700건, 9컬럼)
└── .gitignore
```

## 📊 데이터 컬럼

| 컬럼 | 설명 |
|------|------|
| 번호 | 일련번호 |
| contentId | 한국관광공사 콘텐츠 ID |
| 제목 | 관광지명 |
| 카테고리ID | 콘텐츠 유형 코드 |
| 카테고리명 | 관광지/문화시설/음식점 등 |
| 개요 | 장소 설명 |
| 주소 | 도로명 주소 |
| **지역명** | 시·도 단위 지역명 (18개) |
| **감정** | 키워드 기반 감정 분류 (13종) |

## 🚀 실행 방법

```bash
pip install streamlit pandas plotly

streamlit run app.py
```
